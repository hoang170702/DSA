# ADD TWO NUMBER

## Vấn đề
````
1. Input: cho hai list node l1 và l2
2. Output: kết quả của l1 + l2
````

## Ví dụ
````
Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]
Explanation: 342 + 465 = 807.
Example 2:

Input: l1 = [0], l2 = [0]
Output: [0]
Example 3:

Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Output: [8,9,9,9,0,0,0,1]
````

## Giải quyết vấn đề

### Cách 1 duyệt qua hai LinkedList rồi cộng chúng đưa vào một LinkedList mới:
````
1. Khai báo một Node giả
2. Khai báo một Node current để duyệt và xây dựng danh sách kết quả, nghĩa là current sẽ chứa kết quả của node giả trả về 
3. Khai báo một số nguyên int carry dùng để chứa kết quả số dư khi cộng vượt 9
4. Chạy vòng lắp while, điều kiện là l1 != null || l2 != null || số dư không phải là 0
5. cho 2 biến val1, val2 chứa kết quả trả về của l1 và l2, nếu như null thì cho = 0
6. cho thêm 1 biến sum để chưa kết quả của val1 + val2 + carry
7. dùng carry = sum /10 để giữ lại số dư hàng chục phía trước
8. dùng sum%10 để lấy được kết quả số đơn vị phía sau
9. check nếu như l1 != null thì cho nó băng l1.next
10. check nếu như l2 != null thì cho nó băng l2.next
11. kết thúc vòng lặp trả về Node giả.next để lấy kết quả (next ở đây là toàn bộ danh sách phía sau node giả, ko phải là một giá trị)
````
