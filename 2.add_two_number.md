# ADD TWO NUMBER

## Vấn đề
````
1. Input: cho hai list node l1 và l2
2. Output: kết quả của l1 + l2
````

## Ví dụ
````
Input: l1 = [2,4,3], l2 = [5,6,4]
Output: [7,0,8]
Explanation: 342 + 465 = 807.
Example 2:

Input: l1 = [0], l2 = [0]
Output: [0]
Example 3:

Input: l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
Output: [8,9,9,9,0,0,0,1]
````

## Giải quyết vấn đề

### Cách 1 duyệt qua hai LinkedList rồi cộng chúng đưa vào một LinkedList mới:
````
1. Khai báo một Node giả = = new ListNode(0);
2. Khai báo một Node current để duyệt và xây dựng danh sách kết quả, nghĩa là current sẽ chứa kết quả của node giả trả về 
3. Khai báo một số nguyên int carry dùng để chứa kết quả số dư khi cộng vượt 9
4. Chạy vòng lắp while, điều kiện là l1 != null || l2 != null || số dư không phải là 0
5. check nếu như l1 != null -> sum += l1.val -> l1 = l1.next
6. check nếu như l2 != null -> sum += l2.val -> l2 = l2.next
7. carry sẽ chia 10 để chắc chắn là vòng lặp vẫn sẽ tiếp tục cho đến khi nó = 0
8. currentNode.next sẽ = new ListNode(sum % 10) -> %10 để lấy số hàng đơn vị
9. current sẽ = currentNode.next để chứa kết quả tiếp the
10. kết thúc vòng lặp trả về Node giả.next để lấy kết quả (next ở đây là toàn bộ danh sách phía sau node giả, ko phải là một giá trị)
````

````
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode fakeNode = new ListNode(0);
        ListNode currentNode = fakeNode;
        int carry = 0;

        while(l1 != null || l2 != null || carry != 0){
           int sum = carry;

            if(l1 != null){
                sum += l1.val;
                l1 = l1.next;
            }

            if(l2 != null){
                sum += l2.val;
                l2 = l2.next;
            }

            carry = sum / 10;
            currentNode.next = new ListNode(sum % 10);
            currentNode = currentNode.next;
        }

        return fakeNode.next;
    }
}
````
