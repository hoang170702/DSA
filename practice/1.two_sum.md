# TWO SUM

## Vấn đề
````
1. Input: cho một mảng các số nguyên NUMS và một kết quả TARGET
2. Output: trả về vị trí của 2 phần tử trong mảng NUMS, nếu chúng cộng lại bằng TARGET
````
## Ví dụ
````
Ví dụ 1:
Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Bởi vì nums[0] + nums[1] == 9, chúng ta trả về [0, 1].

Ví dụ 2:
Input: nums = [3,2,4], target = 6
Output: [1,2]

Ví dụ 3:
Input: nums = [3,3], target = 6
Output: [0,1]
````
## Giải quyết vấn đề

### Cách 1: độ phức tạp O(n2)
````
1. Cho chạy hai vòng for ((i = 0) và (j = i + 1)) < nums.length
2. Lấy nums[i] + nums[j] cho đến khi bằng target thì return i,j
````

````
class Solution {
    public int[] twoSum(int[] nums, int target) {
        for(int i = 0; i< nums.length; i++){
            for(int j = i + 1; j < nums.length; j++){
                if(nums[i] + nums[j] == target){
                   return new int[] {i, j};
                }
            }
        }
        return new int[]{};
    }
}
````

### Cách 2 dùng hash map, chạy một vòng for O(n)

````
1. Tạo 1 hash map rỗng
2. Tạo 1 vòng for for (i = 0) < nums.length
3. Tạo một biến temp = target - nums[i]
4. Kiểm tra nếu hash map có key = temp thì return map.get(temp), i
5. Nếu hash map chưa có key = temp thì đưa giá trị hiện tai (key), vị trí (value) vào hash map, map.push(nums[i],i)


==> liên tục lấy target - cho nums[i] để tìm temp, check nếu trong hash map đã có temp thì get temp và vị trí hiện tại của nums[i] ra,
    nếu không phải thì push key,value - num[i],i vào hash map để đợi check cho vòng lặp sau
````

````
class Solution {
    public int[] twoSum(int[] nums, int target) {

        HashMap<Integer, Integer> map = new HashMap<Integer, Integer>();

        for(int i = 0; i< nums.length; i++){
            int temp = target - nums[i];
            if(map.containsKey(temp)){
                return new int[]{map.get(temp),i};
            }
            map.put(nums[i],i);
        }
        return new int[]{};
    }
}
````
